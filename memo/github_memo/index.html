GithubとBootstrapでお手軽ホームページ作成

2020/08/15

僕のホームページはGithubとBootstrapを利用して作られています。Githubでは、htmlファイルをアップロードするだけでホームページを公開できます。 Bootstrapを利用すれば、cssやjsを書く必要もありません！ということで、GithubとBootstrapを使ったホームページの作成手順をまとめてみました。

はじめに

僕のホームページの作成手順をまとめました。
ここで紹介するホームページの作り方は、Github使いにとってはとても簡単な手順ですが、 「Githubって何？」「Github使ったことない！」という方には少々難しいかもしれません、、、。
また、Github Pagesは静的なウェブサイトのみをサポートしています。CMSの利用やウェブアプリの作成はできませんのでご注意ください。


STEP1:Githubのアカウントを作る

「Githubを使ったことがない」という方はこちらでアカウントを作りましょう。
STEP2以降はGithubを使ったことがある前提で話を進めますので、

こちらのQiitaの記事等でGithubアカウントの作り方やGithubの使い方を学びましょう。


STEP2:"ユーザ名".github.ioという名前のレポジトリを作る

"ユーザ名".github.ioという名前のレポジトリを作りましょう。例えば、ユーザ名がmatyalatteならmatyalatte.github.ioという名前のレポジトリを作ります。 公開設定をPublicにしないと無料でホームページを公開できませんので注意してください。レポジトリの作り方はSTEP1で紹介したqiitaの記事を参考にしてください。


STEP3:作成したレポジトリにindex.htmlをアップロードする

内容は何でも良いのでindex.htmlという名前のhtmlファイルを作成してアップロードしましょう。とりあえず僕のホームページのテンプレートをコピペしてindex.htmlというファイル名で保存してアップロードするとよいでしょう。


STEP4:Github PagesのSourceをmasterに設定

レポジトリのフロントページから「Settings」をクリックします。

画面を下の方にスクロールすると「GitHub Pages」という項目があります。「Source」のBranchを「master」に設定してください(おそらくデフォルトではNoneが選択されています)。 ここで「masterを選択できない」「なんか支払い画面がでた」という場合はレポジトリの公開設定がPrivateになっていると思われます。 一つ下の項目「Danger Zone」の「Change repository visibility」から公開設定をPublicに変更してください。


STEP5:ブラウザで"ユーザ名".github.ioにアクセス！

これであなたのホームページが公開されました！ブラウザのURLにhttps://"ユーザ名".github.ioを入力して確認してみましょう。 こんな感じでhtmlファイルが読み込まれているはずです。あとは好きなように自分のホームページをデザインしましょう。

※ホームページの変更が反映されるまで若干時間がかかります(たぶん1分くらい)。htmlファイルが読み込まれない場合は少し時間をおいてから再度読み込みましょう。


Bootstrap CDNでデザインを変えてみよう

Bootstrap CDNを利用すればcssやjsを用意する必要はありません！htmlファイルのみで簡単にページのデザインを記述できます。 僕のホームページのテンプレをコピペした方は、すでにBootstrapのcssとjsを読み込む設定になっているため、bodyタグ内の記述だけでBootstrapを利用できます。
ここではBootstrapの詳しい説明をしませんが、有用な外部サイトを紹介します。参考にしてみてください。
「どうしてもBootstrapの使い方がわからない」あるいは「使い方を勉強するのが面倒くさい」という方は僕のホームページのhtmlファイルを丸パクリしても構いません(ただし、文章や画像は差し替えてください)。
